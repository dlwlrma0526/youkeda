## 绑定本地仓库到GitHub
1. 在GitHub创建仓库
1. 进入文件夹
1. 将本地文件夹绑定到仓库

``` git init 初始化为git操纵库```

``` git remote add origin 仓库地址 将本地文件夹绑定到远程仓库```

4. 提交工程
